Intérieur et Mise en Scène

This is my first ever coding project. I decided to host the site on Github due to it’s popularity, I also decided to use Visual Studio Code due to it’s popularity (I’ve been using these ever since). Using git has been an adventure, and even though this is a personal project with only one developer (me!) I try to use good git practices just to make sure that I’m getting into good habits. I prefer to learn by doing and this project helped me to fully grasp the foundations of HTML and CSS. Thanks to this project I’ve learnt the basics of how to display my first HTML page. Bootstrap helped me to build quickly and with good CSS practices (it’s been great for learning). I’ve used their jumbotron, navbar and carousel components, which I often go back to for inspiration in my other projects. I had a lot of fun putting together the styles and theming for the website, particularly the font selection. Other fun parts of the site are the onHover CSS transitions for the gallery images, as well as a fun “masonry” package that I used to jigsaw together images on the “ambiance” page. This also used a little bit of JavaScript that tracks the users scroll positions and only shows images to them once they can be visible on the page; it does this with a simple fade in animation once the class changes.

I really enjoyed building this website because there a solution for everything you want to build or you’re being asked to build and I found it very intellectually challenging to learn all the new techniques and technology that I’ve leveraged. I had several meetings with the interior designer to come up with the look and feel of the site, which was a lot of fun. I built this site in my spare time whilst holding down a full-time job. It was difficult for me to juggle both at the same time because I wanted to spend more time working on the site than I did at my marketing job! It was around this time when I started to consider that web development might be a possible career path for me.
