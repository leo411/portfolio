Films-App

This is a technical test. The web-app Films app lets you browse among 130 000 films and find the one you're looking for in milliseconds thanks to the powerful AlgoliaSearch API . You also have the possibility to add a movie to the index by clicking the button 'Add a movie' at the top of the page you will then need to fill a form (all the fiels are required). üéûÔ∏èüìΩÔ∏èüé¨üé• This app is made of two parts an Express backend handling a http routing and a api endpoint and a searchPage using the Algolia Javascript API client method. The instantSearch library was great to use. Algolia provides many widgets that make the front-end looks good. I've built a form so the user can post a movie to the backend.

Then the Express back-end send a request to the Algolia backend. The Algolia back-end send a response (success/fail) to the Express backend. The response is displayed on the front-end thanks to an alert message. It was great to learn a bit of Node and Express along the way.
