Weather-App

This is a personal project. The challenge was to build a SPA using an API. I've used an API from openweathermap.org. I've used React and its hooks to build the app. I've used the useState and useEffect hooks. I'm aware that the fetch is not supported by every browsers but i've used it anyway because it is the more modern way. The other challenge was to use Typescript. I've found it very nice to work with because it helps you debug the app very early on.
